Chat App. : WebSocket + Stomp

Application de Chat en utilisant WebSocket en ajoutant STOMP par dessus.
Stomp
STOMP (Simple Text Oriented Messaging Protocol) est un protocole simple basé sur du texte, conçu pour fonctionner avec un middleware orienté message (MOM).
Tout client STOMP peut communiquer avec n'importe quel MessageBroker (courtier de messages) STOMP et être interopérable entre les langages et les plates-formes.
